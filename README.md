# OOP Lab Repository

Đây là kho lưu trữ các bài tập và chương trình của môn Lập trình hướng đối tượng.

## Cấu trúc thư mục

- `LibraryApp/`
  - Chứa mã nguồn cho ứng dụng quản lý thư viện.
  - Bao gồm các file lớp `Book`, `BorrowerRecord`, `Library` và `main.cpp` để chạy chương trình.
- `Review/`
  - Chứa bài tập ôn tập các khái niệm từ Lab 1.
  - Đây là phần luyện tập để củng cố kiến thức qua các ví dụ và bài toán đơn giản.
- `main.cpp`
  - File chính ở root có thể là chương trình mẫu hoặc bài thực hành độc lập.

## Hướng dẫn

1. Mở dự án trong môi trường phát triển C++.
2. Dùng trình biên dịch `g++` hoặc công cụ đã cài đặt để biên dịch từng file `main.cpp` tương ứng.
3. Chạy chương trình và kiểm tra kết quả đầu ra.

## Lưu ý

- Nếu thấy file `main.exe`, đó có thể là file thực thi đã tạo sẵn.
- `Review/` là phần dành cho ôn tập, không phải dự án chính của thư viện.

---

## Mục tiêu

- Tổng hợp lại các khái niệm Lab 1.
- Thực hành lập trình OOP với C++.
- Hiểu cách thiết kế lớp và tổ chức mã nguồn theo mô-đun.
